## M5 Unit Encoderの動作確認用プログラムです。

I2Cの制御(PORT.A)で、Buttonの状態の取得と表示、Encoderの値の取得と表示、
操作に応じたLEDの点灯と消灯を行います。

- ボタン押す：左右のLEDを青に点灯する
* 右へ回す：右のLEDを赤に点灯する
+ 左へ回す：左のLEDを緑に点灯する
